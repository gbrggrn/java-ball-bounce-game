# Ball Bounce Game

## Context

* **Origin** Developed for fun to practice java and basic physics based rendering.
* **Objective** This game lets you bounce a ball against a surface that changes tilt.
* **Status** 🟢 Complete/Functional

---

## Systems Architechture

* **Logic** GameEngine + GUI + Listeners + Player model
* **Tech Stack** Java + Java Swing

---

## Functionality

* Ball bouncing with basic physics
* High score logging + view

---

## Setup & Usage

1. Clone the repository
2. Open in IDE of your choice
3. Bounce some balls and beat the highscore!

---

## Learning Outcomes

* Java Swing GUI elements
* Java syntax
* Basic physics based rendering and collision logic
